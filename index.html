Facial Landmarking Model trained on a dlib 81-point face landmark dataset
Summer 2019 • Facial Recognition • Machine Learning • dlib

Languages/Tools Used: Python, dlib, iMutils, OpenCV, Scipy

This is a Python script which utilizes a dlib 81-point face landmark dataset to facially landmark 81 locations of a face. The dlib library is used to first detect the faces and the dat file is used for the landmarks. An extra touch to the script was the use of counting how many blinks were made. This was done via the Ear-Aspect-Ratio(EAR) which makes use of the 6 coordinates surrounding the eyes.
